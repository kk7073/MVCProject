## 📌 기능 소개
### 주요 기능
|기능|내용|
|--|------|
|**상품 주문 기능**|원하는 상품을 주문하는 기능.|
|**주문 내역 확인**|주문한 상품에 내역을 확인하는 기능|
|**상품 관리 기능**|관리자로 상품을 추가하고 삭제하고 수정할 수 있는 기능|

### 상세 기능
- 로그인, 로그아웃 및 회원가입
- 회원 조회 기능
